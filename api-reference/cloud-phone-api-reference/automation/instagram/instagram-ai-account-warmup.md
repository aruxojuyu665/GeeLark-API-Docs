# Instagram AI account warmup

> API Reference > Cloud Phone API Reference > Automation > Instagram > Instagram AI account warmup

**Source:** [https://doc.geelark.com/web/#/602813392/101528405](https://doc.geelark.com/web/#/602813392/101528405)

---

## Request URL

- `https://openapi.geelark.com/open/v1/rpa/task/instagramWarmup`

## Request Method

- POST

## Request Parameters

| Parameter | Required | Type | Description |
| --- | --- | --- | --- |
| name | No | string | Task name, up to 128 characters |
| remark | No | string | Remarks, up to 200 characters |
| scheduleAt | Yes | int | Scheduled time (timestamp) |
| id | Yes | string | Cloud phone ID |
| browseVideo | No | int | Number of videos viewed, 1-100 |
| keyword | No | string | Search keyword |

## Request Example

```json
{
 "name":"test",
 "remark":"test remark",
 "scheduleAt": 1741846843,
 "id":"557536075321468390",
 "browseVideo":1
}

```
