# Facebook auto login

> API Reference > Cloud Phone API Reference > Automation > Facebook > Facebook auto login

**Source:** [https://doc.geelark.com/web/#/602813392/101528374](https://doc.geelark.com/web/#/602813392/101528374)

---

## Request URL

- `https://openapi.geelark.com/open/v1/rpa/task/faceBookLogin`

## Request Method

- POST

## Request Parameters

| Parameter | Required | Type | Description |
| --- | --- | --- | --- |
| name | No | string | Task name, up to 128 characters |
| remark | No | string | Remarks, up to 200 characters |
| scheduleAt | Yes | int | Scheduled time (timestamp) |
| id | Yes | string | Cloud phone ID |
| email | Yes | string | Email, up to 64 characters |
| password | Yes | string | Password, up to 64 characters |

## Request Example

```json
{
 "name":"test",
 "remark":"test remark",
 "scheduleAt": 1741846843,
 "id":"557536075321468390",
 "email":"test@gmail.com",
 "password": "123456"
}

```
