# Open the app on Google for browsing

> API Reference > Cloud Phone API Reference > Automation > Google > Open the app on Google for browsing

**Source:** [https://doc.geelark.com/web/#/602813392/101528472](https://doc.geelark.com/web/#/602813392/101528472)

---

## Request URL

- `https://openapi.geelark.com/open/v1/rpa/task/googleAppBrowser`

## Request Method

- POST

## Request Parameters

| Parameter | Required | Type | Description |
| --- | --- | --- | --- |
| name | No | string | Task name, up to 128 characters |
| remark | No | string | Remarks, up to 200 characters |
| scheduleAt | Yes | int | Scheduled time (timestamp) |
| id | Yes | string | Cloud phone ID |
| appName | Yes | string | app name |
| description | No | string | describe your experience |

## Request Example

```json
{
  "name":"test",
  "remark":"test remark",
  "scheduleAt": 1741846843,
  "id":"557536075321468390",
  "appName":"TikTok",
  "description": "good app"
}

```
