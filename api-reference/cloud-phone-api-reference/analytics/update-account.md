# Update Account

> API Reference > Cloud Phone API Reference > Analytics > Update Account

**Source:** [https://doc.geelark.com/web/#/602813392/101529393](https://doc.geelark.com/web/#/602813392/101529393)

---

## Request URL

- `https://api.geelark.cn/open/v1/analytics/accounts/update`

## Request Method

- POST

## Request Parameters

| Parameter | Required | Type | Description | Example |
| --- | --- | --- | --- | --- |
| id | Yes | string | Account ID | 565523829426802069 |
| account | No | string | Platform account, maximum 64 characters | myAccount |
| channel | No | int | Platform | 0: TikTok 1: YouTube 2: Instagram |
| remark | No | string | Remark | myRemark |

## Request Example

```json
{
    "id": "565523829426802069",
    "account": "abc"
}

```

## Response Example

```json
{
    "traceId": "9D1A84D6A2A8F9A8A5CD9BA7B7E84281",
    "code": 0,
    "msg": "success"
}

```
