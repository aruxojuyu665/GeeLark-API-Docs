# Launch browser

> API Reference > Browser API Reference > Browser Management > Launch browser

**Source:** [https://doc.geelark.com/web/#/602813392/101528633](https://doc.geelark.com/web/#/602813392/101528633)

---

## Interface Description
Used to start the browser, you need to specify the environment ID.

## Request URL

- `http://localhost:40185/api/v1/browser/start`

## Request method

- POST

## Request Parameters

| Parameter Name | Required | Type | Description | Example |
| --- | --- | --- | --- | --- |
| id | Yes | string | browser id | 539893235657500146 |

## Request Example

```json
{
 "id": "539893235657500146"
}

```

## Example response

```json
{
  "traceId": "123456ABCDEF",
  "code": 0,
  "msg": "success"
}

```
