# Create automated tasks

> User Guide > Cloud Phone > Create automated tasks

**Source:** [https://doc.geelark.com/web/#/602813392/101527835](https://doc.geelark.com/web/#/602813392/101527835)

---

**Create Account Management Task:** Directly call the [Add Task API](https://open.geelark.com/web/#/602813392/101527843).
**Create Video or Image Collection Posting Task:** First, upload the materials, then call the [Add Task API](https://open.geelark.com/web/#/602813392/101527843).

## Creating Video or Image Collection Posting Task

- [File Upload](https://open.geelark.com/web/#/602813392/101527836)
- Use the `resource access URL` obtained in step 1 as the upload URL for the video or image, and then call the [Add Task API](https://open.geelark.com/web/#/602813392/101527843).
